# Política de Privacidad de AI-Buddy

Última actualización: 03/12/2025

Tu privacidad es importante. Este documento explica cómo AI-Buddy maneja tus datos en su nueva versión.

## 1. Recopilación y Uso de Datos

El bot AI-Buddy no mantiene una base de datos de usuarios ni de conversaciones. Su funcionamiento se basa exclusivamente en la ejecución de comandos puntuales.

- **Sin Historial de Chat:** A diferencia de versiones anteriores, el bot ya NO accede, lee ni almacena los últimos mensajes del canal para obtener contexto. Únicamente procesa el texto exacto que introduces dentro de un comando (por ejemplo: `/search` o `/joke`).

- **Procesamiento Efímero:** Los datos introducidos se cargan únicamente en la memoria temporal (RAM) durante el instante que tarda en generarse la respuesta. Una vez enviada, estos datos se descartan automáticamente y no quedan guardados en ningún servidor ni registro (log).

## 2. Compartición con Terceros

Para generar las respuestas y realizar búsquedas, el contenido de los comandos se envía a los siguientes proveedores:

- **Groq (IA):** Procesa el texto para generar la respuesta inteligente.
- **DuckDuckGo (Búsquedas):** Si utilizas el comando de búsqueda, tu consulta se envía de forma anónima para obtener resultados de la web.

No compartimos tus datos con ninguna otra entidad ni los vendemos.

## 3. Retención de Datos

No almacenamos nada. Dado que no guardamos historiales de chat, IDs de usuarios ni preferencias en ninguna base de datos persistente, no existe un registro histórico que podamos consultar o borrar pasado el momento de la interacción.
