# Política de Privacidad de AI-Buddy

Última actualización: 25/11/2025

Tu privacidad es importante. Este documento explica cómo AI-Buddy maneja tus datos.

## 1. Recopilación y Uso de Datos
El bot AI-Buddy no mantiene una base de datos de usuarios ni de conversaciones. Su funcionamiento se basa exclusivamente en el procesamiento en tiempo real.

- Historial de Chat (Contexto): Al recibir un comando o mención, el bot accede temporalmente a los últimos 10 mensajes del canal para dotar de contexto a la Inteligencia Artificial y mantener la coherencia de la conversación.

- Procesamiento Efímero: Estos mensajes se cargan únicamente en la memoria temporal (RAM) durante el instante que tarda en generarse la respuesta. Una vez enviada la respuesta, estos datos se descartan automáticamente y no quedan guardados en ningún servidor ni registro (log) del desarrollador.

## 2. Compartición con Terceros (IA)
Para generar las respuestas, el contenido de esos últimos mensajes se envía a la API del proveedor de Inteligencia Artificial Groq/Google Gemini.

- Dicho proveedor procesa el texto para generar la respuesta.

- No compartimos tus datos con ninguna otra entidad ni los vendemos.

## 3. Retención de Datos
No almacenamos nada. Dado que no guardamos historiales de chat, IDs de usuarios ni preferencias en ninguna base de datos persistente, no existe un registro histórico que podamos consultar o borrar pasado el momento de la interacción.
